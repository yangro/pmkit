## Quick Start

### Option A: Claude Projects (claude.ai)

1. Download the `.md` file from the `skills/` folder
2. Open [Claude Projects](https://claude.ai)
3. Create or open a project
4. Click **Project Knowledge** → **Add content** → Upload the file
5. Paste your doc and ask Claude to "simulate stakeholder feedback"

### Option B: Claude Code (CLI)

1. Download the skill file
2. Add to your project's `.claude/skills/` folder
3. Claude will automatically discover and use it
